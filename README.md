## 🛠️ Troubleshooting

| ❌ Problem | ✅ Fix |
|---|---|
| Mods don't apply | Run `ModPackManager.exe` as Administrator |
| No FPS improvement | Disable overlays (Discord, GeForce Experience, Steam) and retry |
| Visual glitches / artifacts | Reset Visual Enhancer to defaults, update GPU drivers |
| Game won't launch | Restore backup via the backup tool, re-apply mods one at a time |
| Still stuttering | Enable both **Stutter Eliminator** + **VRAM Optimizer** together |
| HUD elements missing | Open HUD Tweaker, reset to default layout |
